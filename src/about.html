the website

i made this site not only to get better at web-dev, but also because the idea of having a personal webspace appealed to me. this was the first big web project i made, meant to be maintained for a long time — it's done wonders for my coding skill, honestly. with that improvement, this site has of course improved since its conception; this is the third recode, but the layout's essentially the same. perhaps i just like it too much?

the main accent colors of this site are pantone's rose quartz and serenity. the background is a f2u pattern i found from a now-deactivated tumblr blog. for credits and such, check the colophon!
